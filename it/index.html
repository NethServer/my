    Stack Tecnologico¶

    • Backend: Go 1.24+ con framework Gin
    • Database: PostgreSQL con migrazioni
    • Cache: Redis per caching ad alte prestazioni
    • Identity: Logto per autenticazione e RBAC
    • Frontend: Vue.js 3 (vedi repository separato)
